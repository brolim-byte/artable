# Artable 

Plataforma web para conectar clientes com ilustradores talentosos. Transforme suas ideias em ilustrações únicas!

## Sobre o Projeto

O **Artable** é um site que oferece serviços de ilustração personalizada, incluindo retratos, fanarts e artes customizadas. A plataforma permite que os clientes visualizem portfólios de ilustradores e solicitem orçamentos de forma fácil e intuitiva.

## Funcionalidades

- **Página Inicial**: Apresentação do serviço com design moderno e responsivo
- **Portfólio de Ilustradores**: Visualização dos artistas disponíveis e seus trabalhos
- **Solicitação de Orçamento**: Formulário completo para solicitar orçamentos personalizados
- **Modo Escuro/Claro**: Alternância entre temas para melhor experiência do usuário
- **Sistema de Avaliação**: Avaliação com estrelas no formulário de orçamento
- **Design Responsivo**: Adaptado para diferentes tamanhos de tela

## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilização e design responsivo
- **JavaScript**: Funcionalidades interativas e modo escuro
- **Google Fonts**: Fonte Poppins

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/brolim-byte/artable.git
```

2. Abra o arquivo `index.html` em seu navegador

3. Navegue pelas páginas:
   - Página inicial: `index.html`
   - Portfólio: `ilustradores.html`
   - Solicitar orçamento: `tabela.html`

## Modo Escuro

O site possui um botão para alternar entre modo claro e escuro, localizado no menu de navegação. Todos os textos e elementos foram otimizados para garantir boa legibilidade em ambos os modos.
